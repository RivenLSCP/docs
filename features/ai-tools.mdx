## Getting Started

### Quick Start

```python
from sera.tools import AIToolkit
from sera.wallet import SeraWallet

# Initialize with wallet for payments
toolkit = AIToolkit(
    wallet=SeraWallet.from_private_key("0x..."),
    config={
        "default_storage_epochs": 12,
        "min_stake": 100  # KYNE tokens
    }
)

# Deploy model with revenue sharing
deployment = await toolkit.deploy_model(
    model_path="./my_model",
    pricing={
        "inference": 1,      # KYNE per request
        "revenue_share": 0.1 # 10% to contributors
    },
    compute_config={
        "gpu": "a100",
        "min_instances": 1
    }
)

# Monitor earnings
stats = await deployment.get_stats()
print(f"Total revenue: {stats.revenue} KYNE")
print(f"Active users: {stats.unique_users}")
```

### Training Pipeline

<Steps>
  1. **Reserve Resources**
     ```python
     # Stake tokens for training
     reservation = await toolkit.reserve_training(
         gpu_type="a100",
         duration_hours=24,
         stake_amount=1000  # KYNE tokens
     )
     ```

  2. **Train Model**
     ```python
     # Start training with marketplace integration
     training_job = await toolkit.train_model(
         dataset_id="0x...",  # From marketplace
         config={
             "share_checkpoints": True,  # List on marketplace
             "revenue_share": 0.1       # To dataset providers
         }
     )
     ```

  3. **Deploy & Monetize**
     ```python
     # Deploy with token incentives
     deployment = await training_job.deploy(
         pricing={
             "requests": 1,    # KYNE per request
             "subscription": { # Monthly access
                 "price": 100,
                 "min_stake": 50
             }
         }
     )
     ```
</Steps>
