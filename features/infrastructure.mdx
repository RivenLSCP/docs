## Storage System

### Walrus Integration

```python
from sera.storage import WalrusClient

# Initialize storage client
storage = WalrusClient(network="mainnet")

# Store data
cid = storage.store(
    data_path="./my_dataset",
    redundancy=3,
    encryption=True
)

# Retrieve data
storage.retrieve(
    cid="Qm...",
    destination="./retrieved_data"
)
```

### Usage Example

```python
from sera.vector import VectorDB

# Initialize vector database
db = VectorDB.connect(cluster="main")

# Create collection
collection = db.create_collection(
    name="embeddings",
    dimension=768
)

# Add vectors
collection.add(
    vectors=embeddings,
    metadata=metadata
)

# Query
results = collection.query(
    vector=query_vector,
    top_k=10
)
```
