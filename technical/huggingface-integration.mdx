## Getting Started

### Prerequisites

<Steps>
  1. **HuggingFace Account**
     Create an account at [HuggingFace](https://huggingface.co)

  2. **Access Token**
     Generate an access token for private resources

  3. **Sera SDK**
     Install the Sera SDK:
     ```bash
     pip install sera-ai[huggingface]
     ```
</Steps>

## Model Integration

### Importing Models

```python
from sera.integrations.huggingface import ModelImporter

# Initialize importer
importer = ModelImporter(
    hf_token="your_huggingface_token"
)

# Import model
model_id = importer.import_model(
    repo_id="bert-base-uncased",
    framework="pytorch",
    task="text-classification"
)

# Import with custom configuration
model_id = importer.import_model(
    repo_id="gpt2",
    config={
        "quantization": "int8",
        "device_map": "auto",
        "max_memory": {0: "10GB"}
    }
)
```

## Dataset Integration

### Importing Datasets

```python
from sera.integrations.huggingface import DatasetImporter

# Initialize dataset importer
importer = DatasetImporter()

# Import dataset
dataset_id = importer.import_dataset(
    repo_id="squad",
    split="train",
    cache_dir="./cache"
)

# Import with transformations
dataset_id = importer.import_dataset(
    repo_id="squad",
    transforms=[
        ("select_columns", {"columns": ["question", "answer"]}),
        ("rename_column", {"old": "question", "new": "input"}),
        ("filter", {"column": "input", "min_length": 10})
    ]
)
```

## Advanced Features

### Custom Processing

```python
from sera.integrations.huggingface import Pipeline

# Define custom processing pipeline
pipeline = Pipeline()
    .add_step("tokenize", tokenizer="bert-base-uncased")
    .add_step("encode", max_length=512)
    .add_step("format", output_format="torch")

# Import with pipeline
dataset_id = importer.import_dataset(
    repo_id="squad",
    pipeline=pipeline
)
```
