## Overview

Sera provides native desktop applications that offer a powerful interface for managing AI data assets, running models, and interacting with the platform's features. Available for Windows, macOS (both ARM64 and x64), and Linux.

### Platform-Specific Notes

<CardGroup cols={2}>
  <Card title="Windows Installation" icon="windows">
    ```bash
    # Verify hash (PowerShell)
    Get-FileHash sera-windows.exe -Algorithm SHA256
    
    # Install silently
    sera-windows.exe /S
    ```
  </Card>
  <Card title="macOS Installation" icon="apple">
    ```bash
    # Verify hash
    shasum -a 256 Sera.dmg
    
    # Install via CLI
    sudo hdiutil attach Sera.dmg
    cp -R "/Volumes/Sera/Sera.app" /Applications/
    ```
  </Card>
</CardGroup>

## Integration

### API Access

```python
from sera.desktop import DesktopClient

# Connect to desktop app
client = DesktopClient()

# Manage datasets
dataset = client.import_dataset(
    path="./local_dataset",
    name="my_dataset",
    description="Local dataset import"
)

# Deploy model
deployment = client.deploy_model(
    model_path="./local_model",
    resources={
        "gpu": 1,
        "memory": "4GB"
    }
)
```

### Automation

```python
from sera.desktop.automation import Workflow

# Create automated workflow
workflow = Workflow("data_processing")
workflow.add_step("import_data", source="./data")
workflow.add_step("preprocess", config={"format": "parquet"})
workflow.add_step("upload", destination="sera://datasets")

# Schedule workflow
workflow.schedule(
    cron="0 0 * * *",  # Daily at midnight
    notify=True
)
```
