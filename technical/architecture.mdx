### Logging & Tracing

```python
from sera.monitoring import Tracer

# Initialize tracer
tracer = Tracer(service_name="compute-service")

# Create span
with tracer.start_span("process-job") as span:
    span.set_tag("job_id", job_id)
    # Process job
    span.log_kv({"event": "job_completed"})
```
