## Data Management

### Uploading Data

```python
from sera.upload import DataUploader

# Initialize uploader
uploader = DataUploader(
    asset_type="dataset",
    privacy_level="private"
)

# Upload with metadata
asset_id = uploader.upload(
    path="./my_dataset",
    metadata={
        "description": "Training dataset for NLP",
        "format": "parquet",
        "size": "10GB",
        "schema": schema_definition
    }
)
```
