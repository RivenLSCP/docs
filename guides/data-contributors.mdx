## Installation Steps

<Steps>
  1. **Install Chrome Extension**
     Visit the [Chrome Web Store](https://chrome.google.com/webstore) and install the Sera extension

  2. **Connect Wallet**
     Click the Sera icon and connect your preferred crypto wallet

  3. **Configure Settings**
     Set your preferred privacy and data sharing options

  4. **Start Contributing**
     Begin using ChatGPT as normal - the extension will handle the rest
</Steps>

## Maximizing Rewards

### Reward Formula Explained
```math
Total Reward = T * S^q

Where:
T = Data Volume Coefficient
S = Session Volume
q = Quality Multiplier
```

<Note>
Early contributors receive higher rewards due to the decreasing Data Volume Coefficient.
</Note>
