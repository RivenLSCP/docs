## Quick Start Guide

<Steps>
  1. **Connect Wallet**
     - Install SUI wallet (Sui Wallet or OKX Wallet)
     - Get some SUI for gas (~0.1 SUI to start)
     - Connect wallet to start earning
  
  2. **Install Tools**
     - Get our Chrome extension
     - Enable AI chat capture
     - Configure earning preferences
  
  3. **Start Earning**
     - Chat with supported AI models
     - Earn SERA tokens per quality conversation
     - Get bonus rewards for premium models
</Steps>

### Bonus Multipliers

Your final reward is calculated as:
```python
reward = base_rate * quality_score * model_tier * (1 + bonuses)

# Example
base_rate = 10 SERA
quality_score = 0.8    # Based on quality checks
model_tier = 1.5      # Premium model
bonuses = 0.3         # Staking + streaks
final_reward = 15.6 SERA
```

Earn from your network:
- Get 5% of referred users' rewards for 30 days
- Earn bonus multipliers for active referrals

- Up to 100 active referrals

```python
# Example referral earnings
referral_reward = friend_earnings * 0.05
```
