## Installation

### Browser Extension

1. **Access the Chrome Web Store**: Navigate to the [Chrome Web Store](https://chrome.google.com/webstore).
2. **Search for Sera**: Enter "Sera AI-To-Earn" in the search bar.
3. **Add Extension**: Click on "Add to Chrome" to install the extension.
4. **Setup**: Follow the on-screen instructions to connect your wallet.
